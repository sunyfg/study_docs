# MySQL

## 安装

```sh
# 下载地址
https://dev.mysql.com/downloads/mysql/
```

- Server Only
- Standalone
- port: 3306
- Authentication Method - 选择 Use Legacy Authentication

## SQL 语言分类

- DML：添加、修改、删除、查询
- DCL：用户、权限、事务
- DDL：逻辑库、数据表、视图、索引

### SQL语句注意事项

- SQL语句不区分大小写，但是字符串区分大小写
- SQL语句必须已分号结尾
- SQL语句中的空白和换行没有限制，但是不能破坏语法
